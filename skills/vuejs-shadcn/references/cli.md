## init

Initialize project configuration and dependencies.

```bash
pnpm dlx shadcn-vue@latest init
```

### Options

| Option | Description |
|--------|-------------|
| `-y, --yes` | Skip confirmation prompt |
| `-f, --force` | Force overwrite existing configuration |
| `-c, --cwd <path>` | Working directory (default: current) |
| `-s, --silent` | Mute output |
| `-b, --base-color <color>` | Base color: neutral, gray, zinc, stone, slate |
| `--css-variables` | Use CSS variables for theming (default: true) |
| `--no-css-variables` | Use utility classes for theming |
| `--src-dir` | Use src directory structure |

### Interactive Prompts

```
Which framework are you using? › Vite / Nuxt / Laravel
Which color would you like to use as base color? › Neutral
Where is your global CSS file? › src/assets/index.css
Do you want to use CSS variables for colors? › yes
Where is your tailwind.config.js located? › tailwind.config.js
Configure the import alias for components: › @/components
Configure the import alias for utils: › @/lib/utils
```

### Generated Files

- `components.json` - Project configuration
- `src/lib/utils.ts` - Utility functions (cn)
- CSS variables in global CSS file

## add

Add components to your project.

```bash
# Add single component
pnpm dlx shadcn-vue@latest add button

# Add multiple components
pnpm dlx shadcn-vue@latest add button card dialog

# Add all components
pnpm dlx shadcn-vue@latest add --all

# Add from URL (custom registry)
pnpm dlx shadcn-vue@latest add https://example.com/r/my-component.json
```

### Options

| Option | Description |
|--------|-------------|
| `-y, --yes` | Skip confirmation prompt |
| `-o, --overwrite` | Overwrite existing files |
| `-c, --cwd <path>` | Working directory |
| `-p, --path <path>` | Custom path for component |
| `-a, --all` | Add all available components |
| `-s, --silent` | Mute output |
| `--nodep` | Skip installing dependencies |

### Examples

```bash
# Add with auto-confirm
pnpm dlx shadcn-vue@latest add button -y

# Overwrite existing component
pnpm dlx shadcn-vue@latest add button -o

# Add to custom path
pnpm dlx shadcn-vue@latest add button -p ./src/components/custom

# Add all without prompts
pnpm dlx shadcn-vue@latest add --all -y
```

### Component Selection

When run without arguments, shows interactive selection:

```
Which components would you like to add? › Space to select. Return to submit.
◯ accordion
◯ alert
◯ alert-dialog
◯ aspect-ratio
◯ avatar
◯ badge
◯ button
◯ card
...
```

## update

Update components in your project.

⚠️ **Warning:** This overwrites modifications. Commit changes first.

```bash
# Update single component
pnpm dlx shadcn-vue@latest update button

# Update multiple components
pnpm dlx shadcn-vue@latest update button card dialog

# Update all components
pnpm dlx shadcn-vue@latest update
```

### Options

| Option | Description |
|--------|-------------|
| `-c, --cwd <path>` | Working directory |
| `-y, --yes` | Skip confirmation prompt |

## build

Generate registry JSON files for custom component registries.

```bash
pnpm dlx shadcn-vue@latest build
```

### Options

| Option | Description |
|--------|-------------|
| `-c, --cwd <path>` | Working directory |
| `-o, --output <path>` | Output directory (default: public/r) |

### Usage

1. Create `registry.json` in project root:

```json
{
  "$schema": "https://shadcn-vue.com/schema/registry.json",
  "name": "my-registry",
  "homepage": "https://example.com",
  "items": []
}
```

2. Add components to `registry/` directory
3. Run build command
4. Serve from `public/r/[component].json`

## components.json

Configuration file generated by `init`.

```json
{
  "$schema": "https://shadcn-vue.com/schema.json",
  "style": "default",
  "typescript": true,
  "tailwind": {
    "config": "tailwind.config.js",
    "css": "src/assets/index.css",
    "baseColor": "neutral",
    "cssVariables": true,
    "prefix": ""
  },
  "framework": "vite",
  "aliases": {
    "components": "@/components",
    "utils": "@/lib/utils",
    "ui": "@/components/ui",
    "lib": "@/lib",
    "hooks": "@/hooks"
  },
  "iconLibrary": "lucide"
}
```

### Configuration Options

| Key | Description |
|-----|-------------|
| `style` | Component style variant |
| `typescript` | Use TypeScript |
| `framework` | vite, nuxt, laravel |
| `tailwind.config` | Path to tailwind config |
| `tailwind.css` | Path to global CSS |
| `tailwind.baseColor` | Base color palette |
| `tailwind.cssVariables` | Use CSS variables |
| `tailwind.prefix` | Tailwind class prefix |
| `aliases.components` | Components import alias |
| `aliases.utils` | Utils import alias |
| `aliases.ui` | UI components directory |
| `iconLibrary` | Icon library (lucide, radix) |
